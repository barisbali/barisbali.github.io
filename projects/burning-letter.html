BURNING LETTER

Vanish your emotions from the world forever

ABOUT THIS PROJECT

Write a letter when you feel like it, pour your words and emotions into it and burn it with a tap, vanish it from the world forever! This project is an interactive web experience serving as an emotional outlet, wrapped in a polished interface and animations. Includes achievements and statistics with lots of eastereggs to make your experience more enjoyable.

KEY FEATURES

  • Immersive, cathartic interaction design.
  • Custom "burning" visual effects.
  • Client-side processing (no data is stored, true privacy).
  • Smooth, physics-based UI transitions.
  • Achievements and Easter eggs.
