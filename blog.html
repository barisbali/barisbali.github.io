Inspiration

Today I want to share my experience building this retro-cyberpunk themed portfolio website. The inspiration came from my love for 80s aesthetics and cyberpunk culture.

Design Decisions

When approaching the design, I focused on several key elements:

  • Neon color palette inspired by synthwave aesthetics
  • Grid-based background reminiscent of 80s sci-fi
  • Retro-futuristic typography using pixel-style fonts

Technical Challenges

Here's an example of the grid background implementation:

.grid-background {
    background: 
        linear-gradient(var(--grid-color) 1px, transparent 1px),
        linear-gradient(90deg, var(--grid-color) 1px, transparent 1px);
    background-size: 40px 40px;
    transform: perspective(500px) rotateX(60deg);
}
